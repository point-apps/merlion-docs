## List all captures {{ tag: 'GET', label: '/v1/captures' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of all your captures. By default, a maximum of ten captures are shown per page.

    ### Optional attributes

    <Properties>
      <Property name="page" type="integer">
        The page number of captures returned.
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of captures returned.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/captures">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/captures', {
      params: {
        filter: {
          page:1,
          pageSize: 10,
          search: '',
          isDraft: true,
        },
        sort: 'date-'
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
    ```json {{ title: '200' }}
    {
      "captures": [
        {
          "_id": "637b569149e0c02e1036c35a",
          "activity": "Catur",
          "clusters": [
            {
              "name": "A"
            },
            {
              "name": "B"
            }
          ],
          "isDraft": false,
          "date": "2022-01-01T00:00:00.000Z",
        },
      ],
      "pagination": {
        "page": 1,
        "pageSize": 1,
        "pageCount": 1,
        "totalDocument": 2
      }
    }
    ```
    </CodeGroup>

## Assign permissions {{ tag: 'PATCH', label: '/v1/captures/:id/assign-permissions' }}

<Row>
  <Col>

    This endpoint allows you to perform an update on a capture to assign permission.

    ### Optional attributes

    <Properties>
      <Property name="permissions" type="array<string>">
        The array of permission name for the capture.
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="PATCH" label="/v1/captures/637d83d15d2be122007524bf">
    ```typescript {{ title: 'axios' }}
    const response = axios.patch('/v1/captures/637d83d15d2be122007524bf', {
      permissions: [
        {
          name: 'create-user',
          assign: true
        },
        {
          name: 'read-user',
          assign: false
        }
      ]
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
    ```json {{ title: '204' }}
    // empty response
    ```
    </CodeGroup>
  </Col>
</Row>
