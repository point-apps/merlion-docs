# Ikigai

Ikigai report from captures activity

## Get ikigai report {{ tag: 'GET', label: '/v1/ikigai-report' }}

<Row>
  <Col>

    This endpoint allows you to retrieve a paginated list of ikigai report based on authenticated user.
    
    ### Required attributes

    <Properties>
      <Property name="dateStart" type="datetime">
        The starting date for filtering report
      </Property>
      <Property name="dateEnd" type="datetime">
        The ending date for filtering report
      </Property>
      <Property name="ikigai" type="array<object>">
        Filtering report based on ikigai type
      </Property>
      <Property name="ikigai.type" type="string">
        Ikigai type `enjoy` `easy` `excelent` `earn`
      </Property>
    </Properties>

    ### Optional attributes

     <Properties>
      <Property name="page" type="integer">
        The page number of captures returned.
        - default `1`
      </Property>
      <Property name="pageSize" type="integer">
        The page size number of captures returned.
        - default `10`
      </Property>
    </Properties>

  </Col>
  <Col sticky>
    <CodeGroup title="Request" tag="GET" label="/v1/captures">
    ```typescript {{ title: 'axios' }}
    const response = axios.get('/v1/report/ikigai', {
      params: {
        filter: {
          dateStart: '',
          dateEnd: '',
          ikigai: [
            {
              type: 'easy',
              value: true
            },
            {
              type: 'excelent',
              value: false
            },
          ]
        }
      }
    })
    ```
    </CodeGroup>
    <CodeGroup title="Response">
    ```json {{ title: '200' }}
    {
      "typology": ['evaluating', 'analysing'],
      "pagination": {
        "page": 1,
        "pageSize": 1,
        "pageCount": 1,
        "totalDocument": 2
      }
    }
    ```
    </CodeGroup>
  </Col>
</Row>
